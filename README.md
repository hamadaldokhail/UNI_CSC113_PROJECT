# Hotel Reservation System

Java console-based hotel reservation system for CSC 113.

## What it does
Manages hotel data for:
- Rooms
- Guests
- Reservations

## Features
- Add / Remove / Search Rooms
- Add / Remove / Search Guests
- Make / Cancel / Search Reservations
- Display All Reservations
- Count available rooms (recursion)

## OOP concepts used
- Inheritance (single and multi-level)
- Abstract class (`Reservation`)
- Interface (`Searchable`)
- Polymorphism
- Arrays of objects
- Recursion

## How to run
Run `HotelReservationApp` and use the menu.
